## 📌 Overview

**Terraform EKS CI/CD Platform** is an end-to-end cloud-native DevOps implementation that automates infrastructure provisioning, application deployment, security validation, and monitoring on AWS.

The project demonstrates a production-style DevOps lifecycle using:

- Infrastructure as Code (Terraform)
- Kubernetes orchestration (Amazon EKS)
- CI/CD automation (Jenkins)
- Containerization (Docker)
- Application packaging (Helm)
- Artifact management (Nexus)
- Code quality analysis (SonarQube)
- Security scanning (CodeQL & Veracode)
- Monitoring and observability (Prometheus & Grafana)
